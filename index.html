📱 Πρόσβαση μέσω QR Code

Για Καθηγητές:
Μετά την ανάρτηση των αρχείων σε web server, μπορείτε να δημιουργήσετε QR codes για κάθε δραστηριότητα:

  1. Ανεβάστε τα αρχεία σε GitHub Pages ή Netlify
  2. Πάρτε το URL κάθε δραστηριότητας
  3. Χρησιμοποιήστε QR Code Generator
  4. Εκτυπώστε τα QR codes για την τάξη

💡 Tip: Κάθε μαθητής μπορεί να σκανάρει και να κάνει τις δραστηριότητες στο κινητό του!
